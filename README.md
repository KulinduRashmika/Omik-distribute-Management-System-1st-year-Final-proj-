## âš™ï¸ How to Run the Project (XAMPP)

1. Install **XAMPP** on your computer.
2. Start **Apache** and **MySQL** from the XAMPP Control Panel.
3. Copy the project folder into the `htdocs` directory inside the XAMPP installation folder.
4. Open **phpMyAdmin** (`http://localhost/phpmyadmin`).
5. Create a new database (e.g., `omik_db`).
6. Import the provided `.sql` file into the database.
7. Update database connection details in the PHP config file if needed.
8. Open a browser and go to:

   ```
   http://localhost/Omik-distribute-Management-System
   ```
